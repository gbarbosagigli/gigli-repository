Capstone Project: Winning the Space Race with Data Science

This project focuses on analyzing SpaceX launch data to understand key factors influencing launch success rates, payload patterns, and costs. Using Python, Pandas, Matplotlib, and Dash, I created visualizations to uncover insights and built predictive models to estimate the likelihood of launch success.

Key Findings:

  • Identified key variables correlated with successful launches, such as booster reuse and payload type.
  • Analyzed cost trends to evaluate SpaceX's competitive advantage over competitors.
  • Built a logistic regression model to predict future launch outcomes with over 80% accuracy.
